Detour

	The Detour product redirects calls to another object or URL. With default
	settings, you only need to specify an id for the object and a destination URL,
	such as "http://www.example.com". The default settings are: no prefix, 301
	status code and no URL truncatating.

	You can optionally specify
	- an absolute or relative url to redirect to
	- if to prefix the url with a dynamic url
	- if to remove or append any sub paths part of the request
	- the HTTP status code
